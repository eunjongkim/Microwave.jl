# Microwave
[Julia](julialang.org) library for calculation with Microwave objects.

Written by Eun Jong Kim (ekim7206@gmail.com)

## Usage
```jl
Pkg.clone("https://github.com/eunjongkim/Microwave.jl.git")
using Microwave
```
